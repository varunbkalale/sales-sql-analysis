# 📊 Sales Performance Analysis (SQL Project)

## 📌 Overview
This project analyzes sales data using MySQL to uncover key business insights related to revenue, profitability, and regional performance.

---

## 🛠 Tools Used
- MySQL

---

## 📁 Project Structure
- data/ → dataset  
- queries/ → SQL queries  
- docs/ → detailed project explanation  

---

## 🔍 Key Analysis

- Revenue & Profit Analysis  
- Regional Performance Analysis  
- Product Performance  
- Customer Analysis  
- Budget vs Actual Comparison  
- Profit Margin Analysis  

---

## 💡 Key Insights

- Laptop category drives highest revenue  
- Desktop category has highest profitability  
- Bangalore is the top-performing region  
- Revenue is concentrated among few products  

---

## ▶️ How to Use

1. Import dataset into your SQL environment  
2. Run queries from `queries/sales_analysis.sql`  
3. Analyze outputs to derive insights
